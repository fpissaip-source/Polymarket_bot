Every prediction on Polymarket is represented by **outcome tokens**. When you trade, you‚Äôre buying and selling these tokens. Your **position** is simply your balance of tokens for a given market.

## [‚Äã](https://docs.polymarket.com/concepts/positions-tokens\#outcome-tokens)  Outcome Tokens

Each market has exactly two outcome tokens:

| Token | Redeems for | If‚Ä¶ |
| --- | --- | --- |
| **Yes** | $1.00 | The event occurs |
| **No** | $1.00 | The event does not occur |

Tokens are **ERC1155** assets on Polygon, using the [Gnosis Conditional Token Framework](https://github.com/gnosis/conditional-tokens-contracts/) (CTF). This means they‚Äôre fully onchain and function as standard ERC1155 tokens.

Outcome tokens are always fully backed. Every Yes/No pair in existence is
backed by exactly `$1` of USDC.e collateral locked in the CTF contract.

### [‚Äã](https://docs.polymarket.com/concepts/positions-tokens\#split)  Split

Convert USDC.e into outcome tokens. Splitting $1 creates 1 Yes token and 1 No token.

Copy

Ask AI

```
$100 USDC.e ‚Üí 100 Yes tokens + 100 No tokens
```

Use this when you want to:

- Create inventory for market making
- Obtain both sides of a market

### [‚Äã](https://docs.polymarket.com/concepts/positions-tokens\#trade)  Trade

Buy or sell tokens on the order book. This is how most users acquire positions.

- **Buy Yes** at `$0.60` ‚Üí Pay `$0.60`, receive 1 Yes token
- **Sell Yes** at `$0.60` ‚Üí Give up 1 Yes token, receive `$0.60`

You can sell your position at any time before resolution.

### [‚Äã](https://docs.polymarket.com/concepts/positions-tokens\#merge)  Merge

Convert a complete set of tokens back into USDC.e. Merging requires equal amounts of Yes and No tokens.

Copy

Ask AI

```
100 Yes tokens + 100 No tokens ‚Üí $100 USDC.e
```

Use this when you want to:

- Exit a position without trading
- Convert accumulated tokens back to collateral

### [‚Äã](https://docs.polymarket.com/concepts/positions-tokens\#redeem)  Redeem

After a market resolves, exchange winning tokens for USDC.e.

| Outcome | Yes tokens | No tokens |
| --- | --- | --- |
| Event occurs | Worth $1 each | Worth $0 |
| Event doesn‚Äôt occur | Worth $0 | Worth $1 each |

Copy

Ask AI

```
100 winning tokens ‚Üí $100 USDC.e
```

### [‚Äã](https://docs.polymarket.com/concepts/positions-tokens\#position-value)  Position Value

The value of your position depends on the current market price:

```
Position value = Token balance √ó Current price
```

If you hold 100 Yes tokens and Yes is trading at $0.75:

Copy

Ask AI

```
Position value = 100 √ó $0.75 = $75
```

## [‚Äã](https://docs.polymarket.com/concepts/positions-tokens\#profit-and-loss)  Profit and Loss

Your profit depends on how the market resolves compared to your entry price.

### [‚Äã](https://docs.polymarket.com/concepts/positions-tokens\#example-buying-yes-at-0-40)  Example - Buying Yes at 0.40

| Scenario | Outcome | Return | Profit |
| --- | --- | --- | --- |
| Event occurs | Yes wins | $1.00 | +$0.60 per token (150%) |
| Event doesn‚Äôt occur | No wins | $0.00 | -$0.40 per token (-100%) |

### [‚Äã](https://docs.polymarket.com/concepts/positions-tokens\#holding-rewards)  Holding Rewards

Polymarket pays a **4.00% annualized** Holding Reward based on your total position value in eligible markets. Your total position value is randomly sampled once each hour, and the reward is distributed daily. The rate is variable and subject to change at Polymarket‚Äôs discretion.

### [‚Äã](https://docs.polymarket.com/concepts/positions-tokens\#example-selling-before-resolution)  Example - Selling Before Resolution

You can lock in profits or cut losses by selling before the market resolves:

- Bought Yes at `$0.40`
- Price rises to `$0.70`
- Sell at `$0.70` ‚Üí Profit of `$0.30` per token (75%)
